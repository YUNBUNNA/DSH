អ្នកជំនាញអាហារូបត្ថម្ភ & សុខភាពចិត្ត • ថ្ងៃទី០៥ ខែកក្កដា ២០២៤

កីឡាករទាំងអាជីព និងស្ម័គ្រចិត្ត ត្រូវការតុល្យភាពល្អរវាងសុខភាពរាងកាយ និងសុខភាពផ្លូវចិត្ត។ អាហារត្រឹមត្រូវមិនមែនគ្រាន់តែជួយបង្កើនការអនុវត្តកីឡាប៉ុណ្ណោះទេ ប៉ុន្តែថែមទាំងមានឥទ្ធិពលយ៉ាងខ្លាំងទៅលើ អារម្មណ៍ ការផ្តោតអារម្មណ៍ និងសុខុមាលភាពផ្លូវចិត្តទាំងមូល។

ភាពតភ្ជាប់រវាងអាហារ និងសុខភាពចិត្ត

សារធាតុចិញ្ចឹមប៉ះពាល់ដល់អារម្មណ៍

រាងកាយផលិតអរម៉ូនសេរ៉ូតូនីន (serotonin) និងដូប៉ាមីន (dopamine) ពីអាហារដែលយើងទទួលទាន។ អរម៉ូនទាំងនេះគ្រប់គ្រងអារម្មណ៍ ការគេងលក់ និងការផ្តោតអារម្មណ៍។ ការខ្វះសារធាតុចាំបាច់អាចនាំឱ្យមានអារម្មណ៍អាក្រក់ និងនឿយហត់។

ថាមពលស្ថិរភាពសម្រាប់សមត្ថភាព

កម្រិតស្ករក្នុងឈាមមិនស្ថិរធ្វើឱ្យអ្នកមានអារម្មណ៍កោះក្រហម រម៉ៃក់រខាត់ និងមិនអាចផ្តោតអារម្មណ៍។ ការទទួលទានអាហារត្រឹមត្រូវជួយរក្សាថាមពលឱ្យស្ថិរភាពពេញមួយថ្ងៃ ដែលចាំបាច់សម្រាប់ការហាត់ប្រាណ និងការងារប្រចាំថ្ងៃ។

ការពារពីស្ត្រេស និងកង្វល់

អាហារដែលមានសារធាតុប្រឆាំងអុកស៊ីតកម្ម វីតាមីន និងសារធាតុរ៉ែ ជួយការពាររាងកាយ ពីផលប៉ះពាល់នៃស្ត្រេសពីការហាត់ប្រាណ។ វាជួយកាត់បន្ថយការឈឺចុក និងលើកទឹកចិត្ត ដំណើរការស្ទុះសាច់ដុំកាន់តែប្រសើរ។

ផែនការអាហារប្រចាំថ្ងៃ សម្រាប់កីឡាករ

អាហារពេលព្រឹក (៦:០០-៧:០០ព្រឹក)

មុនហាត់ប្រាណពេលព្រឹក:

  • បាយសំប៉ែត ១ ពែង + ស៊ុត ២-៣ គ្រាប់ + ចេក ១ គ្រាប់
  • នំប័ុងធញ្ញជាតិពេញ ២ ដុំ + បឺរ + យ៉ាអួ

ផ្តល់កាបូអ៊ីដ្រាតគ្រប់គ្រាន់សម្រាប់ថាមពលហាត់ប្រាណ

អាហារសម្រន់ (៩:៣០-១០:០០ព្រឹក)

បន្ទាប់ពីហាត់ប្រាណ:

  • ស្មូធី ប្រូតេអ៊ីន (ចេក + យ៉ាអួ + ទឹកដោះគោ + ប្រូតេអ៊ីនម្សៅ)
  • ស៊ុតឆ្អិន ២ គ្រាប់ + នំប័ុង + ផ្លែឈើ

ជួយស្ទុះសាច់ដុំ និងបំពេញថាមពលវិញ

អាហារថ្ងៃត្រង់ (១២:០០-១:០០ថ្ងៃត្រង់)

អាហារធំបំផុត:

  • បាយសំប៉ែត ១-១.៥ ពែង + សាច់ក្រលេញ/ត្រី ១០០-១៥០ក្រាម
  • បន្លែឆ្អិន ១ ពែង + សម្ល
  • ផ្លែឈើតាមរដូវកាល

ផ្តល់សារធាតុចិញ្ចឹមគ្រប់គ្រាន់សម្រាប់ថ្ងៃសកម្ម

អាហារសម្រន់រសៀល (៣:០០-៤:០០រសៀល)

មុនហាត់ប្រាណល្ងាច (១ម៉ោងមុន):

  • ចេក + គ្រាប់អាម៉ុង ១ ក្តាប់តូច
  • ដំឡូងឆ្អិន/ដំឡូងបាត៉ាត + ទឹក
  • បារថាមពលធម្មជាតិ (គ្រាប់ធញ្ញជាតិ + ផ្លែឈើ)

ថាមពលស្រាលងាយស្រួលរំលាយ

អាហារពេលល្ងាច (៦:៣០-៧:៣០ល្ងាច)

បន្ទាប់ពីហាត់ប្រាណ (ក្នុងរយៈពេល ២ម៉ោង):

  • បាយ/មី + ត្រី/សាច់មាន់ + បន្លែច្រើន
  • ដំឡូង + សាឡាត់ + ប្រូតេអ៊ីន
  • សម្ល + ផ្លែឈើបន្តិចបន្តួច

ជួយស្ទុះសាច់ដុំ និងរៀបចំរាងកាយសម្រាប់គេងលក់

អាទិភាពលើការគេងលក់

គេង ៨-១០ ម៉ោងក្នុងមួយយប់សម្រាប់កីឡាករ។ ការគេងលក់គ្រប់គ្រាន់ជួយស្ទុះសាច់ដុំ ពង្រឹងប្រព័ន្ធភាពសុំារ និងលើកកម្ពស់សុខភាពចិត្ត។ បង្កើតទម្លាប់គេងទៀងទាត់។

បង្កើតគាំទ្រសង្គម

រក្សាទំនាក់ទំនងជាមួយគ្រួសារ មិត្តភក្តិ និងសមាជិកក្រុម។ ចែករំលែកបញ្ហា និងជោគជ័យ។ ការមានគាំទ្រសង្គមល្អជួយកាត់បន្ថយស្ត្រេស និងបង្កើនការលើកទឹកចិត្ត។

អនុវត្តការសមាធិ និងយូហ្គា

ស្វែងរកពេលវេលា ១០-២០ នាទីជារៀងរាល់ថ្ងៃសម្រាប់សមាធិ ការដកដង្ហើមយឺតៗ ឬយូហ្គា។ វាជួយរលាយស្ត្រេស បង្កើនការផ្តោតអារម្មណ៍ និងធ្វើឱ្យចិត្តស្ងប់។

ផឹកទឹកគ្រប់គ្រាន់

ផឹកទឹក ៣-៤ លីត្រ/ថ្ងៃ។ បន្ថែមបរិមាណក្នុងថ្ងៃហាត់ប្រាណ និងអាកាសធាតុក្តៅ។

តាមដានវឌ្ឍនភាព

កត់ត្រាអាហារ ការហាត់ប្រាណ និងអារម្មណ៍។ វាជួយអ្នកយល់ពីអ្វីដែលល្អបំផុតសម្រាប់អ្នក។

ការជៀសវាងអាហារូបត្ថម្ភមិនសមស្រប

ការព្រមានសំខាន់

កីឡាករមួយចំនួនប្រឈមនឹងបញ្ហា៖

  • ការគ្រប់គ្រងទម្ងន់មិនសមរម្យ: ការកាត់បន្ថយអាហារខ្លាំងពេក ឬការបង្ខំខ្លួនហាត់ប្រាណលើសកម្រិត
  • ការប្រើអាហារបន្ថែមមិនត្រឹមត្រូវ: ការពឹងផ្អែកលើអាហារបន្ថែមដោយមិនពិគ្រោះអ្នកជំនាញ
  • ការរំលោភបំពានអាហារ: បញ្ហាទាក់ទងនឹងទំនាក់ទំនងអាហារមិនសុខភាព

ចងចាំ: សុខភាពល្អជាងជ័យជំនះ។ ប្រសិនបើអ្នកមានបញ្ហាទាក់ទងនឹងអាហារ ឬការហាត់ប្រាណ សូមស្វែងរកជំនួយវិជ្ជាជីវៈភ្លាមៗ។
